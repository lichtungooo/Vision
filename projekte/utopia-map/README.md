# Utopia Map

Timos Vision einer Map-zentrierten Anwendung auf dem Real Life Stack.

## Kernidee

Die **Map ist IMMER die Basis**. Alle Module öffnen sich als **Floating Windows** auf der Map.

```
┌─────────────────────────────────────────┐
│           MAP (Hintergrund)             │
│                                         │
│  ┌──────────────┐   ┌──────────────┐   │
│  │  Kalender    │   │   Profil     │   │
│  │  (Window)    │   │   (Window)   │   │
│  └──────────────┘   └──────────────┘   │
│                                         │
│         (Luft zwischen Fenstern)        │
└─────────────────────────────────────────┘
```

## Features

### Map-Layer-System
- Fantastische Karten (künstlerische Ebenen)
- Weltraumansicht
- Multiple Layer wählbar

### Content auf Map
- Quest-Icons (farbkodiert)
- Event-Icons
- Marktplatz-Icons (Angebote, Ressourcen, Werkzeuge)

### Floating Windows
- Verschiebbar, skalierbar, andockbar
- Desktop: Bis zu 4 Fenster gleichzeitig
- Mobile: Einzelne Fenster oder Dashboard

### Navigation
- "Navigiere dorthin" → Google Maps
- Alle Locations auf Map sichtbar

## Ziel

Menschen **raus aus der App, rein in die Realität** bringen!
