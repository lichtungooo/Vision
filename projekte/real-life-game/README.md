# Real Life Game

Gamification für die echte Welt.

## Kernidee

Lerne neue Fähigkeiten durch **Quests** – real, vor Ort, mit anderen Menschen.

## Dokumente

- [quest-gamification-system.md](quest-gamification-system.md) – Vollständige Spezifikation

## Features

### Quests
- Reale Herausforderungen (Handwerk, Natur, Gemeinschaft)
- QR-Code oder Foto-Verification
- Quest-Reihen (z.B. Carpenter Level 1-10)

### Belohnungen
- XP für Skills
- Items für Avatar
- Level-Ups

### Skill-Tree

6 Hauptkategorien (Vorschlag von Bernd):

1. **Seele** – Empathie, Mitgefühl, Intuition
2. **Geist** – Logik, Kreativität, Problemlösung
3. **Bewusstsein** – Achtsamkeit, Selbstreflexion
4. **Körper** – Handwerk, Sport, Gesundheit
5. **Gemeinschaft** – Zusammenarbeit, Vertrauen
6. **Soziales** – Konfliktlösung, Führung

> "Wenn Seele, Geist und Bewusstsein eins werden, kannst du alles machen" – Bernd

### Avatar
- Visueller Avatar mit Items
- Items als Belohnungen für Quests
- Seltenheitsstufen (Common → Legendary)

## Beispiel-Flow

1. Quest auf Map finden: "Bogenschießen im Wald"
2. Teilnehmen
3. Real durchführen
4. QR-Code scannen
5. XP + Item "Bogenschütze-Abzeichen" erhalten
6. Item auf Avatar ausrüsten
